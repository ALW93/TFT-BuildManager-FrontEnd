# **Teamfight Tactics Build Manager**

## **About**

Build Manager for Teamfight Tactics, Save and Create Builds along with viewing Builds curated from the current Meta.

Designed to fill a need in the TFT App environment where there has been a significant lack of space for regular users to aggregate builds - the goal is to expand the meta beyond what is defined by a small pool of players.

## **Build Status**

TFT Build Manager is an App that is still currently being developed, in it's current state User's may post their builds and view existing content.
